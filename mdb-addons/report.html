实验目的

        学习web基础知识的应用,熟练应用各种html框架，如bootstrap等，以及网站的部署。

实验内容

        实验内容为设计一个。世界著名国家公园的导览网站，主要应用了bootstrap框架，使用了导航条，图片轮播，iframe内嵌网站，锚链接来增加网站的美观度和易用性等。

遇到的问题

        在本次实验中，我遇到了许多的问题，如图片的轮播如何实现，以及相关的位置及样式的调整等，还有各种如图片不显示，内嵌网站的视频清晰度低等问题。

解决问题

        解决图片轮播的样式问题：通过 data 属性：使用 data 属性可以很容易控制轮播（Carousel）的位置。 属性 data-slide 接受关键字 prev 或 next，用来改变幻灯片相对于当前位置的位置。 使用 data-slide-to 来向轮播传递一个原始滑动索引，data-slide-to="2" 将把滑块移动到一个特定的索引，索引从 0 开始计数。 data-ride="carousel" 属性用于标记轮播在页面加载时就开始动画播放。 通过 JavaScript：轮播（Carousel）可通过 JavaScript 手动调用。 解决视频清晰度不足：使用highquality属性值。

总结

        通过本次实验，我能够进一步熟练应用bootstrap等web框架，对html和css的各种标签和功能有了更多的了解与认识，并且学会了如何将网站部署到网络上，不过本次实验仍然存在许多问题，我会在今后的学习中，努力去解决这些问题，以进一步提高自己的能力。
